# Verzeichnis von Verarbeitungstätigkeiten (Art. 30 DSGVO)

verantwortliche Stelle konkret zu belegen.

## 1. Verantwortlicher

- Name / Behörde:

- Datenschutzbeauftragte:

## 2. Bezeichnung der Verarbeitung

Filterung von KI-generierten Texten zur Entfernung personenbezogener
Daten — Anomyze ≥ 2.0 (siehe `pyproject.toml`).

## 3. Zwecke

- Souveräne KI-Nutzung durch die Bundesverwaltung.
- Schutz Betroffener vor unbeabsichtigter Offenlegung.
- Kanal-spezifische Zwecke: GovGPT (intern), IFG (Veröffentlichung),
  KAPA (parlamentarische Anfragen).

## 4. Kategorien der Betroffenen

Siehe DPIA Abschnitt 1.

## 5. Kategorien personenbezogener Daten

KI-Anwendung / Veröffentlichungsplattform / Parlamentsdirektion
weiter.
